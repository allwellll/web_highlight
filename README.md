# Web Highlight Annotator

一个 Manifest V3 浏览器插件，用于在网页上圈选文字高亮、用笔自由勾画、删除标注，并按当前网址保存与恢复。数据会先保存到本地，再异步同步到坚果云 WebDAV，避免影响页面操作流畅度。

## 仓库说明

本仓库是一个面向 Chrome / Edge 的网页标注扩展，核心目标是在不改写原网页 DOM 的前提下，提供文本高亮、下划线、自由画笔、删除、持久化恢复和坚果云 WebDAV 同步能力。项目重点关注复杂网页、公式内容、翻译插件干扰和大量标注场景下的恢复准确性与浏览性能。

## 功能

- 高亮模式下圈选网页文字后弹出颜色菜单，选择颜色后创建高亮。
- 划线模式下圈选网页文字后弹出颜色菜单，选择颜色后创建下划线标注。
- 支持包含公式、数学符号、Unicode 字符的文本高亮。
- 支持配置默认高亮颜色、颜色菜单和高亮快捷键。
- 支持自由笔迹勾画，可调整颜色和笔宽。
- 支持浏览模式，不进行涂画、高亮或划线。
- 支持本地数据索引、过期清理和画笔点压缩，降低长期使用的本地存储占用。
- 支持删除高亮和笔迹。
- 按完整网址保存当前网页标注，下次打开自动恢复。
- 可配置坚果云 WebDAV，将标注 JSON 异步同步到云端。

## 安装

1. 打开 Chrome / Edge 的扩展管理页面。
2. 开启“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本项目目录。

## 使用

1. 点击扩展图标打开控制面板。
2. 默认是“浏览模式”，不会拦截网页原本操作。
3. 选择“高亮模式”，在页面中框选文字，弹出颜色菜单后点击颜色即可高亮。
4. 也可以先选中文字，再按配置的快捷键（默认 `Alt+H`）用默认颜色高亮。
5. 选择“划线模式”，框选文字后选择颜色即可创建下划线。
6. 点击已有高亮或划线区域会弹出“删除”按钮。
7. 选择“画笔模式”，在页面上按住鼠标或触控笔拖动即可绘制。
8. 选择“删除模式”，点击高亮/划线，或点击笔迹附近即可删除。
9. 点击“清空当前网页标注”会删除当前 URL 下的全部高亮、划线和笔迹。

## 坚果云同步配置

1. 在坚果云账号中创建第三方应用密码。
2. 在插件弹窗中填写：
   - WebDAV 目录：默认 `https://dav.jianguoyun.com/dav/web-highlight`
   - 坚果云账号：登录邮箱
   - 应用密码：第三方应用密码
3. 勾选“启用异步同步”并保存。

插件会以当前网页 URL 的 SHA-256 值作为文件名保存 JSON，例如：

```text
https://dav.jianguoyun.com/dav/web-highlight/<sha256>.json
```

## 数据说明

高亮和划线不会改写原网页 DOM。每条文本标注会同时保存全文偏移量、原始选中文本、选区前后文和 DOM 路径锚点；页面重新加载时会优先用精确偏移恢复，偏移失效后再用 DOM 路径与前后文候选评分重新定位，避免翻译插件插入文本或拆分文本节点后把标注恢复到重复文本的错误位置。对于 KaTeX 这类公式渲染，插件会额外保存公式可见层的相对矩形锚点，避免选区落在隐藏 MathML 上导致数据已保存但页面看不到高亮。如果页面文本已被完全翻译成另一种内容，原文不存在时会跳过该标注，避免强行错位渲染。

本地数据默认保留最近 `500` 个页面，并在页面初始化时清理超过 `180` 天未更新的页面数据。弹窗里的“清理过期本地数据”会迁移旧版本索引并立即执行同样的清理规则。清空当前网页标注会移除当前 URL 对应的本地数据。

## 性能策略

- 文本高亮和划线使用独立覆盖层，不改写网页原始 DOM。
- 渲染时使用 `DocumentFragment` 批量插入，并对页面文本只构建一次索引；大量标注恢复时复用同一个文本索引，offset 查询使用二分定位。
- 标注点击使用统一事件委托和按点击坐标命中检测，不为每个高亮矩形绑定监听器。
- 画笔保存为 SVG path 点集，绘制完成后统一持久化，并对保存做短延迟合并。
- 画笔点默认按约 `4px` 最小距离抽稀，单条笔迹最多保留 `3000` 个点，避免长笔迹持续膨胀。
